# MailSorter

### App Description
MailSorter is a program written in Java that takes in Addresses(6 lines at a time) and sorts them in ascending order of ZipCodes using RadixSort

### Concepts Used:
  1. Radix Sort
  2. Interfaces
  3. Object Oriented Programming in Java
